My Books

Posted January 20th 2021

This website contains several descriptions of some of my favorite books. I created this website to give you some basic overviews of the books that I not only consider to be personal favorites of mine, but also educational. I hope you will like my book descriptions and futhermore, I hope you will purchase one of my books from Amazon. Below is my favorite book

The C Programming Language By Dennis M Ritchie and Brian W Kernighan is an excellent book for both new comers and expert programmers. The book is packed with many examples, and yet is small and concise. The book is written by the person that created the language, so there is no better source of information. I highly reccomend this book to anyone who wishes to become a C Programmer.

HTML and CSS is a great book for new comers to web design. This book is packed with lots of examples, and is easy to read for beginners and challening enough for experienced programmers and web designers. It is a must read. Click on the image for more details on how you can purchase is book.

Database systems is my favorite book for learning SQL and the principals of Database design. SQL is the primary language used for talking to databases, and this book explains the language in detail. Whether you are a novice or advanced SQL developer, you will learn a lot from this book.

Java programming could not be easier with this book. This was the first book I ever read when it came to learning Java. I can tell you from experience that I have NEVER read a better book than this one when it came to learning Java. Java is a tough language to learn but this book makes it easy. Click on the book for more details on how you can purchase the book.
